Woocommerce export order by step-india.com
== Description ==
Plugin provides ability to export selected orders in CSV format.
Also maintains status of whether order is exported or not. Hence you can export orders that are not exported using filter in plugin.

== Screenshots ==
1. screenshot1.jpg shows the filter that gets added upon activation of plugin, this filter is to differentiate orders that are exported and not exported.
2. screenshot2.jpg shows the option that gets activated upon activation of plugin, this option triggers the export of selected orders.
